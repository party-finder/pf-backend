# Find-party-backend

Серверная часть приложения для поиска группы людей по игровым интересам

## Technologies

- NestJS
- Mongoose
- Mongodb

## Installation

Установите все зависимости и запустите сервер

```sh
yarn
yarn run start:dev
```

Для продакшена

```sh
yarn
yarn run start
```

## Swagger

Swagger запускается по следующему адресу

```sh
127.0.0.1:3000/api
```
